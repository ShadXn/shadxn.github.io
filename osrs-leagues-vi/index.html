How Early Game Routing Works

1. Add tasks — Click + Task in the toolbar or right-click anywhere on the map to place a task at that spot. Choose from imported league tasks or create custom helper tasks.

2. Place map pins — Each task can have a pin on the map. The gold pulsing pin is your next task. Dashed lines connect tasks in route order.

3. Order your route — Drag the ⠿ handle in the task list to reorder tasks. The map lines and pin numbers update live.

4. Track progress — Click ✓ to complete or ⟳ to skip a task. You can also click pins on the map to complete/skip directly. The progress bar tracks league vs helper completion and total points earned.

5. Navigate — Click 📍 in a task row to zoom the map to that pin.

Pin colors: 🔵 Blue = League task · 🟣 Purple = Helper · 🟡 Gold = Next up · ⚫ Grey = Complete · 🟠 Orange = Skipped

League tasks come from the Task List tab — import tasks there first, then they appear in the search when adding a task here. Already-added tasks are hidden from the search.

Phases are collapsible groups (e.g. "Varlamore", "Karamja"). Click + Phase to create one. Assign tasks to a phase when adding/editing, or drag a task row onto a phase header. Drag the ⠿ handle on a phase header to reorder phases.

Export / Import — ⬇ saves your route as JSON, ⬆ reloads it. You can also import a leaguesplanner.com .txt export — phases and task names are preserved. Both buttons are in the progress bar row.

Customize pin and line colors via the 🎨 button in the toolbar. In fullscreen, use ◫ to toggle the task panel and ⊟ to toggle the top bar.
